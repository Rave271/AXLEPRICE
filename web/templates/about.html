About

What this demo is doing under the hood.

Overview

This project combines a small neural network for numeric valuation with an optional LLM-generated explanation. The goal is a simple, interview-friendly product demo: clean UX, fast feedback, and easy deployment.

  • Price prediction: local Keras MLP downloaded from Hugging Face Hub.
  • Explanation: optional request to the Hugging Face Inference API.

How it works

  1. User submits a vehicle profile.
  2. The backend preprocesses the inputs to match the training pipeline.
  3. The model predicts a log-price, converted back to GBP.
  4. If enabled, an LLM returns 3–4 concise bullet points explaining major drivers.

Predictions are estimates and can be wrong; treat them as guidance.
